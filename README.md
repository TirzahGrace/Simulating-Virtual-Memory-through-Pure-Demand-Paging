# OS - Virtual Memory through Demand Paging

## Commands to use to run the program:
 - Compile: To compile all files use:
  ```bash
     make all
  ```
 - Clean: To clean use:
 ```bash
     make clean
  ```
 - Run: To run use:
  ```bash
     ./master k m f
  ```
		k = no of processes
		m = max no of pages / process
		f = Total no of frames
  - Output: result.txt
